## User specific settings

‚öôÔ∏è To change your defaults settings by my settings you need to follow this steps :

1. Launch VScode on your device.
2. Make Ctrl + Shift + P to access to the command palette and type : user settings.json and press enter.
3. You will arrive at the json file who you can modify options of your IDE.
4. Delete all the content of this file and paste the content of the "settings.json" file of this repository.
5. Your'e done for the options and you can go to the extensions sections.
